# Marketing-and-Data-Analysis
Tools: DBeaver, BigQuery, Google Analytics 4

Description: Analyze Google/Facebook campaign performance and prepare BI
data, including spend metrics, top days by ROMI, conversions by dates and
channels, landing page comparisons, and engagement-purchase correlations.

My Tasks: Identify top-performing campaigns, calculate metric growth, structure
data for BI, compare landing pages, and analyze engagement vs. purchases.

Results: Gained hands-on experience in marketing data analysis, revealing spend
trends, top-performing campaigns, conversion patterns, and actionable insights for
strategy and budget optimization.
